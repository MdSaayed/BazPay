BazPay Documentation

1. Introduction

This React project is built with React Router, Tailwind CSS, and additional libraries for enhanced user experience. Designed for easy customization and scalability.

2. Installation

Prerequisites

  • Node.js (latest LTS recommended)
  • npm or yarn

Steps

cd BazPay
npm install # or yarn install
npm run dev # or yarn run dev

Project runs at http://localhost:3000/

4. Customization

Customizing Components

To customize components, navigate to the src/components/ directory. Each component is modular and can be easily modified.

Customizing Styles

This project uses Tailwind CSS for styling. You can customize the styles by editing the tailwind.config.js file or by adding custom CSS in the src/index.css/ directory.

Adding New Pages

To add new pages, create a new file in the src/pages/ directory and then add the route in the src/routers/Router.jsx directory.

Customizing Navbar Logo, Footer Text, Footer Logo, and Favicon

You can easily customize the following elements:

  • Navbar Logo:
    • Replace the existing logo file at /public/assets/logo.png with your own logo.
    • Update the logo path in the Navbar component if necessary.
  • Footer Text:
    • Edit the footer text in the Footer component located at src/components/footer/.
  • Footer Logo:
    • Replace the existing footer logo file at /public/assets/footer-logo.png with your own logo.
    • Update the logo path in the Footer component if necessary.
  • Favicon:
    • Replace the existing favicon file at /public/favicon.png with your own favicon.
    • Ensure the favicon is linked correctly in the index.html file.

Example of updating the favicon in index.html:

<link rel="icon" type="image/png" href="./assets/favicon.png" />

5. Deployment

If you want to use the ready-to-production dist/ folder, follow these steps:

  1. Run the production build:
    2. npm run build
  2. Deploy dist/ folder:
    • Netlify: Drag and drop dist/ into Netlify dashboard.
    • Vercel: Run vercel deploy inside dist/.
    • FTP/SFTP: Upload dist/ to your hosting server.
    • cPanel: Upload dist/ via File Manager and extract it.
