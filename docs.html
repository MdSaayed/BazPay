React Router Project Documentation

1. Introduction

This React project is built with React Router, Tailwind CSS, and additional libraries for enhanced user experience. Designed for easy customization and scalability.

2. Installation

Prerequisites

  • Node.js (latest LTS recommended)
  • npm or yarn

Steps

cd BazPay
npm install # or yarn install
npm start # or yarn start

Project runs at http://localhost:3000/

4. Deployment

If you want to use the ready-to-production dist/ folder, follow these steps:

  1. Run the production build:
    2. npm run build
  2. Deploy dist/ folder:
    • Netlify: Drag and drop dist/ into Netlify dashboard.
    • Vercel: Run vercel deploy inside dist/.
    • FTP/SFTP: Upload dist/ to your hosting server.
    • cPanel: Upload dist/ via File Manager and extract it.
